# 재건축 추천 요청

Category: Backend → Agent
Content-Type: application/json
Endpoint: /agent/reconstruction/recommend
Main Fields: jobId, callbackUrl, houseId, address, riskLevel, agingRate, accessibility, populationContext, beforeImageUrl
Main Response: accepted, jobId
Method: POST
Purpose: 특정 빈집 재건축 용도/규모/이미지 추천 요청

## Request

```json
{
  "jobId": "RC-JOB-0001",
  "callbackUrl": "http://backend:8080/api/agent/reconstruction-results",
  "houseId": "VH-001",
  "address": "완산동 123-4",
  "riskLevel": "DANGER",
  "agingRate": 92,
  "accessibility": 85,
  "populationContext": {
    "elderlyRate": 31.2,
    "populationDensity": 1820,
    "mainDemand": ["주거", "돌봄", "상업"]
  },
  "beforeImageUrl": "http://backend:8080/uploads/houses/VH-001-before.jpg"
}
```

## Response

```json
{
  "accepted": true,
  "jobId": "RC-JOB-0001"
}
```
