# 정비 우선순위 분석 실행

Category: Front → Backend
Content-Type: application/json
Endpoint: /api/maintenance/analyze
Main Fields: area
Main Response: success, jobId, message
Method: POST
Purpose: 정비 우선순위 분석 작업 실행

## Request

```json
{
  "area": "영천시 전체"
}
```

## Response

```json
{
  "success": true,
  "jobId": "MA-JOB-0001",
  "message": "정비 우선순위 분석을 요청했습니다."
}
```

## Backend 처리

- 빈집 데이터 조회
- 인구 데이터 조회
- 최근 이벤트 조회
- `/agent/maintenance/analyze` 호출
