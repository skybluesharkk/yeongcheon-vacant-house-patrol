# 빈집 정비 우선순위 분석 요청

Category: Backend → Agent
Content-Type: application/json
Endpoint: /agent/maintenance/analyze
Main Fields: jobId, callbackUrl, area, vacantHouses, populationData, recentEvents
Main Response: accepted, jobId
Method: POST
Purpose: 빈집/인구/이벤트 기반 정비 우선순위 분석 요청

## Request

```json
{
  "jobId": "MA-JOB-0001",
  "callbackUrl": "http://backend:8080/api/agent/maintenance-results",
  "area": "영천시 전체",
  "vacantHouses": [
    {
      "houseId": "VH-001",
      "address": "완산동 123-4",
      "x": 12.4,
      "y": 8.7,
      "riskLevel": "DANGER",
      "agingRate": 92,
      "accessibility": 85,
      "recentEventCount": 3
    }
  ],
  "populationData": [
    {
      "areaName": "완산동",
      "populationDensity": 1820,
      "elderlyRate": 31.2
    }
  ],
  "recentEvents": []
}
```

## Response

```json
{
  "accepted": true,
  "jobId": "MA-JOB-0001"
}
```
