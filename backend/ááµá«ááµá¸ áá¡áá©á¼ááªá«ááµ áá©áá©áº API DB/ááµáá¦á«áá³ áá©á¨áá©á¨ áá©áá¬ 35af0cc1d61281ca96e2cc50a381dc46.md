# 이벤트 목록 조회

Category: Front → Backend
Content-Type: -
Endpoint: /api/events
Main Fields: robotId, eventType, severity, resolved, limit(query)
Main Response: eventId, robotId, eventType, severity, summary, confidence, imageUrl, resolved
Method: GET
Purpose: 이상 탐지/실종자 후보/경고 이벤트 목록 조회

## Query Parameter

| 이름 | 타입 | 필수 | 설명 |
| --- | --- | --- | --- |
| --- | --- | ---: | --- |
| robotId | string | X | 특정 로봇 이벤트만 조회 |
| eventType | string | X | 이벤트 타입 필터 |
| severity | string | X | 심각도 필터 |
| resolved | boolean | X | 처리 여부 |
| limit | number | X | 조회 개수 |
