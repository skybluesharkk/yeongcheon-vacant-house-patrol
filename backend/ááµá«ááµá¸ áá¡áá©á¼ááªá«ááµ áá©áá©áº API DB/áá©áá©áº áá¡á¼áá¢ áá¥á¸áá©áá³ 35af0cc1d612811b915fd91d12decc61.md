# 로봇 상태 업로드

Category: Robot → Backend
Content-Type: application/json
Endpoint: /api/robots/{robotId}/status
Main Fields: status, battery, x, y, address, nextDestination, velocity, timestamp
Main Response: success, robotId
Method: POST
Purpose: 로봇 현재 상태, 위치, 배터리 업로드

## Request

```json
{
  "status": "PATROLLING",
  "battery": 78,
  "x": 12.4,
  "y": 8.7,
  "address": "완산동 123-4",
  "nextDestination": "완산동 125-6",
  "velocity": 0.8,
  "timestamp": "2026-05-20T14:35:22"
}
```

## Response

```json
{
  "success": true,
  "robotId": "robot-01"
}
```

## Backend 처리

- `robot_status`에 최신 상태 저장
- `robot_path`에 현재 좌표 누적 저장
- 대시보드 차량 상태, 위치, 순찰 경로, 통계에 반영
