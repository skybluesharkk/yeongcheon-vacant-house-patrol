# 이미지 이상 탐지 요청

Category: Backend → Agent
Content-Type: application/json
Endpoint: /agent/anomaly/analyze-image
Main Fields: jobId, callbackUrl, robotId, imageId, imageUrl, x, y, address, timestamp
Main Response: accepted, jobId
Method: POST
Purpose: 카메라 이미지 기반 이상 상황 탐지 요청

## Request

```json
{
  "jobId": "AN-JOB-0001",
  "callbackUrl": "http://backend:8080/api/agent/anomaly-results",
  "robotId": "robot-01",
  "imageId": 15,
  "imageUrl": "http://backend:8080/uploads/robots/robot-01/images/frame_001.jpg",
  "x": 12.4,
  "y": 8.7,
  "address": "완산동 123-4",
  "timestamp": "2026-05-20T14:35:22"
}
```

## Response

```json
{
  "accepted": true,
  "jobId": "AN-JOB-0001"
}
```

## 비고

결과는 `/api/agent/anomaly-results`로 콜백 전달한다.
