# 로봇 카메라 이미지 업로드

Category: Robot → Backend
Content-Type: multipart/form-data
Endpoint: /api/robots/{robotId}/image
Main Fields: image, x, y, address, timestamp
Main Response: success, robotId, imageId, imageUrl, analysisJobId
Method: POST
Purpose: 로봇별 카메라 이미지 업로드 및 이상 탐지 요청

## Form Data

| 필드 | 예시 |
| --- | --- |
| image | `frame_001.jpg` |
| x | `12.4` |
| y | `8.7` |
| address | `완산동 123-4` |
| timestamp | `2026-05-20T14:35:22` |

## Response

```json
{
  "success": true,
  "robotId": "robot-01",
  "imageId": 15,
  "imageUrl": "/uploads/robots/robot-01/images/frame_001.jpg",
  "analysisJobId": "AN-JOB-0001"
}
```

## Backend 처리

- 이미지 파일 저장
- `robot_image` 저장
- `/agent/anomaly/analyze-image` 호출
- 결과 콜백 수신 후 이벤트 저장
