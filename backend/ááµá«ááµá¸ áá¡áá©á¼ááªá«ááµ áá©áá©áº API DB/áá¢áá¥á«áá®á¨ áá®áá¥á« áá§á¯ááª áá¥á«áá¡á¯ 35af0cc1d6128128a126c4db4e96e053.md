# 재건축 추천 결과 전달

Category: Agent → Backend
Content-Type: application/json
Endpoint: /api/agent/reconstruction-results
Main Fields: jobId, houseId, recommendedUse, buildingScale, style, estimatedCost, expectedReturn, feasibility, reason, images
Main Response: success, houseId
Method: POST
Purpose: 재건축 추천 결과 저장

## Request

```json
{
  "jobId": "RC-JOB-0001",
  "houseId": "VH-001",
  "recommendedUse": "주거 + 상업 복합",
  "buildingScale": "2F ~ 3F",
  "style": "모던 + 지역 특화",
  "estimatedCost": "약 3.2억원",
  "expectedReturn": "연 6.8%",
  "feasibility": "HIGH",
  "reason": "접근성이 높고 주변 인구 밀도가 높아 소규모 주거 및 상업 복합 건물로 재건축하는 것이 적합합니다.",
  "images": {
    "beforeImageUrl": "http://backend:8080/uploads/houses/VH-001-before.jpg",
    "afterImageUrl": "http://agent:8000/uploads/reconstruction/VH-001-after.png",
    "mainImageUrl": "http://agent:8000/uploads/reconstruction/VH-001-main.png"
  }
}
```

## Response

```json
{
  "success": true,
  "houseId": "VH-001"
}
```

## Backend 처리

- `reconstruction_result` 저장
- 대시보드 reconstruction 영역에 반영
