# 이상 탐지 결과 전달

Category: Agent → Backend
Content-Type: application/json
Endpoint: /api/agent/anomaly-results
Main Fields: jobId, analysisId, robotId, imageId, eventType, severity, confidence, summary, detectedObjects
Main Response: success, eventId
Method: POST
Purpose: 이미지 분석 결과를 백엔드에 저장

## Request

```json
{
  "jobId": "AN-JOB-0001",
  "analysisId": "AN-0001",
  "robotId": "robot-01",
  "imageId": 15,
  "eventType": "MISSING_PERSON_CANDIDATE",
  "severity": "URGENT",
  "confidence": 0.87,
  "summary": "완산동 123-4 인근에서 실종자 후보와 유사한 인물이 감지되었습니다.",
  "detectedObjects": [
    {
      "label": "person",
      "confidence": 0.94,
      "bbox": {"x1": 120, "y1": 80, "x2": 260, "y2": 420}
    }
  ],
  "missingPersonCandidate": {
    "matched": true,
    "candidateId": "MP-003",
    "similarity": 0.87
  },
  "recommendedDashboardMessage": "실종자 후보 발견 - 완산동 123-4 인근"
}
```

## Response

```json
{
  "success": true,
  "eventId": 3
}
```

## Backend 처리

- `eventType = NORMAL`이면 저장 생략 가능
- NORMAL이 아니면 `event` 저장
- `MISSING_PERSON_CANDIDATE`는 urgentEvent로 대시보드 노출
