# 이벤트 처리 완료

Category: Front → Backend
Content-Type: application/json
Endpoint: /api/events/{eventId}/resolve
Main Fields: memo
Main Response: success, eventId, resolved
Method: POST
Purpose: 이벤트 확인 완료 처리

## Request

```json
{
  "memo": "관리자 확인 완료"
}
```

## Response

```json
{
  "success": true,
  "eventId": 3,
  "resolved": true
}
```

## Backend 처리

- `event.resolved = true`
- `resolved_memo`, `resolved_at` 저장
- 오늘 처리 완료 통계에 반영
